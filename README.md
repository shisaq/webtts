# Edge TTS Generator

A web app that converts text to natural-sounding speech using Microsoft Edge TTS (via [edge-tts-universal](https://www.npmjs.com/package/edge-tts-universal)).

## Features

- 400+ voices across multiple languages and locales
- Streaming audio playback in the browser
- Download generated audio as MP3

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```
npm run build
npm start
```
